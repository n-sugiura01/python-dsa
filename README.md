# Python DSA Learning Roadmap

データ構造とアルゴリズムの学習用リポジトリです。

## 構成
- **00_intro**: 計算量 (Big O) や Python 特有のテクニック
- **01_data_structures**: 基本的なデータ構造の実装
- **02_algorithms**: ソート、探索などのアルゴリズム
- **03_patterns**: 面接や競プロ頻出の解法パターン
- **tests**: Pytestによるテストコード

## 実行方法
```bash
pip install -r requirements.txt
pytest
```
